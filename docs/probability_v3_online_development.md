# 三位彩在线概率 v3 开发评估

## 目标与规则

本实验验证逐期开奖反馈是否能从14个统计模型中学到稳定权重，不替换概率 v2，也不建立正式前向实验。

1. 前100期只建立基础历史统计。
2. 第101期开始，每期先由均匀基线和14个统计模型分别输出完整 `000-999` 概率。
3. 开奖前按当前权重合并概率并生成直选、组选纯Top10。
4. 开奖后根据各模型给真实开奖号的概率更新权重：`w_i <- w_i * p_i(actual)^eta`。
5. 每次更新后按固定份额向初始先验收缩，避免单期结果让权重永久归零。
6. 温度 `0.2`、学习率 `1.0`、固定份额 `0.01` 和均匀先验权重 `0.5` 在运行前固定，不根据评估段结果调整。

两份官方数据各1000期：前500期用于在线预训练，其中第101至500期产生400次反馈；后500期继续严格执行“先预测、后开奖、再更新”。这500期此前已经用于其他开发评估，不能称为未见验证。

本次在线实现源码指纹为 `a718fc922ae77ac8f45552f3c47a040a6f138332d2f679900ef066fd1fe9f282`。福彩3D、排列三输入 CSV SHA-256 分别为 `1b2c49f4cfaff4484251dcf32b3210f69f6cdb9be64dbcef35fbd15432770654`、`410ee8fb396f94093fe48fc0c513005c7b7d61b8eee8a356dcd9a172b1c09ad3`。

## 决策

- 不根据这500期结果调整温度、学习率或收缩比例后再次宣称通过。
- 不将在线 v3 接入日报主推荐；当前结果只证明反馈机制可运行，没有证明预测优势。
- 已新增独立 `online_probability` 日报模式；它会持久化状态并消费新增开奖，但默认 `ensemble` 日报保持不变，在线 v3 仍需新的未见前向样本才能申请主流程资格。
- 后续若继续研究，应先校准单模型概率或引入与开奖号历史不同源的机制信息，再建立新的未来开奖前实验。
