# 本次自动执行报告

## 需求摘要

- 在只保留福彩3D、排列三、排列五的基础上完善预测验证流程。
- 只有严格前推结果明显且稳定高于精确随机基线时，算法才标记为可行。

## 关键假设

- 不承诺通过调整模型制造统计优势；证据不足时必须输出“不可行”。
- 三位彩默认完整空间评分，历史罕见结构不再硬排除。
- 详见 ASSUMPTIONS.md。

## 实现

- 新增精确随机可行性模块，覆盖 Poisson-binomial 右尾概率、99% Wilson 下界、相对提升和三段稳定性。
- 严格前推逐期记录直选/组选随机覆盖率，报告按统计策略和集成策略分别判定。
- 实盘调权最低样本从 5 期提高到 100 期，改用同候选空间随机率和 `p<0.01` 门槛，权重浮动上限从 20% 降到 10%。
- 福彩3D、排列三默认保留全部 1000 个合法号码进入评分，显式硬过滤仍可用。
- 新增固定白名单的福彩3D/排列三公开历史抓取，抓取后再次使用既有加载器复核。
- 使用两份各1000期真实数据运行最后500期完整严格前推，生成 schema v5 Markdown/JSON 报告。
- 新增显式冻结快照：schema v3 保存候选配置和推荐指纹，普通日报可重复生成，已冻结快照不可被覆盖。
- 以 `2026187` 为截止期登记 `fc3d_ensemble_forward_v1`，初始推荐指纹为 `6d96c9ceede14d7b859aa07ad4556062bd0671c7641a3aee05d072500acfbc9e`。
- 新增概率 v2：完整1000状态概率、14模型剖面消融、独立守门、均匀回退、直选纯Top10和组选排列概率求和。
- 概率快照使用独立 `probability_v2` 实验 ID，避免与相同源期的集成快照和复盘文件冲突。
- 新增在线概率 v3：均匀基线与14个统计模型逐期合并，开奖后按真实号概率更新并保存前后权重。
- `digit-report` 新增 `online_probability` 模式：首次全量重放，后续新增期增量反馈，状态指纹失配自动重建；当前两种彩票日报已实跑并验证重复运行快照幂等。
