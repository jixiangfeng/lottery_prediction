# Changelog

本文件遵循 Keep a Changelog，版本采用 SemVer。

## [Unreleased] - 2026-07-17

### Added

- 完整化 learned_ranker_v4：canonical 独立窗口权重、五类特征 × `30-300/50-all/30÷300` 三种独立趋势变换、全行为搜索空间 manifest、seeded 有界采样和惰性特征准备。
- 新增 common/direct/group 分项闸门、部分启用 activation、日报 `mainRecommendation/research` 分区与 v4 专用不可变实盘复盘/隔离汇总。
- 新增标准化数值期号排序的 `canonicalDataSha256`，贯穿参数、冻结评估、日报、快照和开奖后复盘校验。
- 新增 fc3d/pl3 train→evaluate→daily CLI 集成、跨进程确定性、损坏/伪造/过期证据、实验隔离和组选语义回归测试。

- 新增三位彩固定评分算法 v4：完整 `000-999` 多窗口特征、贝叶斯平滑、时间衰减、遗漏/趋势/上期关系、固定线性评分和稳定 tie-break。
- 新增可复现随机/局部参数搜索，严格隔离 search、validation 和 frozen test，并搜索权重、温度、组选聚合、窗口、半衰期与遗漏上限。
- 新增冻结逐期前推评估，输出 LogLoss、Brier、排名、直选/组选 TopK、逐期精确组选随机基线、单侧 p 值、分块稳定性和闸门结论。
- 新增 `scripts/digit_learned_ranker.py` 的 train/evaluate/daily 子命令，以及参数、评估、日报和不可覆盖快照的 CSV/源码/参数指纹。
- `digit-report` 新增 `learned_ranker_v4` 模式；未通过闸门时明确标记“研究模式，不接入主推荐”。

### Fixed

- 修复 v4 真实号概率/排名通过 `int(号码)` 定位导致候选行顺序变化时结果错误，并使分布指纹及组选/位置概率聚合与候选顺序无关。
- 修复 validation 分数参与局部 search 起点选择、evaluate 重新按当前 CSV 推导冻结切分，以及参数元数据未纳入完整性校验的问题。
- 冻结评估新增完整报告指纹、源码/CSV/参数产物指纹和 test 未参与选参证明；daily 仅在玩法与完整冻结证据全部匹配时允许晋级。
- 修复冻结评估和同源期快照可被部分覆盖、损坏评估导致 daily 崩溃，以及非法 TopK/池大小产生无效随机基线的问题。

- 新增直选/组选统计可行性闸门，输出精确随机期望、Poisson-binomial 单侧 p 值、99% Wilson 下界、相对提升和三段时间稳定性。
- 严格前推 JSON schema 升级到 v5，按 `current_statistics` 和 `ensemble_voting` 分别输出可行性明细。
- 新增显式 `digit-fetch` 命令，使用固定官方域名白名单、分页、超时重试、格式校验和原子写入获取福彩3D/排列三历史。
- 新增两种彩票各 500 期、16 模型完整配置的官方数据严格前推报告；当前所有整体可行性闸门均不通过。
- 新增福彩3D `fc3d_ensemble_forward_v1` 协议，以 `2026187` 为截止期冻结初始候选、算法配置、数据哈希和推荐指纹。
- 推荐快照升级到 schema v3，增加候选配置、SHA-256 推荐指纹和显式不可变标记；普通日报仍允许重复生成。
- 新增三位彩概率 v2：完整空间归一化、模型剖面消融、选参/守门分离、均匀回退、直选纯TopK和组选排列概率求和。
- 新增概率 v2 独立严格前推命令及福彩3D/排列三各500期开发报告；两种彩票校准均失败并回退均匀分布。
- 新增在线概率 v3：均匀基线与14个统计模型逐期开奖后更新权重，保存完整权重轨迹，并完成福彩3D/排列三各500期开发评估。
- `digit-report` 新增 `online_probability` 日报模式：持久化在线状态，新增开奖增量反馈，历史修正自动重建，并保存下一期概率快照。

### Removed

- 移除快乐8玩法注册、数据抓取、统计候选、回测、日报和旧 PyTorch Pipeline。
- 移除快乐8专用 H5、脚本、测试、文档与已提交报告。
- 移除 Copula、图嵌入、规则挖掘等仅服务于已删除玩法的模块和重型依赖。

### Changed

- 项目范围收敛为福彩3D、排列三、排列五。
- Makefile、环境模板、依赖、覆盖率门槛和最小示例改为数字彩专用。
- Python 运行依赖锁定为 NumPy、pandas、SciPy 和 scikit-learn。
- 福彩3D、排列三默认评分空间放宽为完整 `000-999`，和值、跨度、豹子和上期原号不再默认硬排除，显式过滤配置保持可用。
- 实盘模型调权改用同候选数量随机基线，最低样本提高到 100 期，仅对 `p < 0.01` 的正向证据调权，单模型浮动上限降为 10%。
- `digit-report` 增加 `--freeze-pick`，Makefile 对应 `DIGIT_FREEZE_PICK=1`。
- 推荐快照支持 `experiment_id`，概率 v2 与旧集成快照可以同源期并存且复盘文件不互相覆盖。

## [2.0.0] - 2026-07-16

### Added

- 数字彩完整空间候选评分、16 槽位集成排序和多随机基线前推。
- 福彩3D、排列三组选与直选候选，排列五向量化直选评分。
- 版本化增量统计快照、理论概率枚举和并发原子写。
- 开奖前候选留痕、后续自动复盘和模型级保守调权。
